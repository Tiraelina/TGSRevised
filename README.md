# TGSRevised map WCSharp source
This map was originally some version of The Great Strategy v2 by Hank_s. It's been tossed around as a community project for our now bi-weekly game night for almost 20 years that I believe started off as fixing memory leaks. Then significant re-balancing and an ability system that actually had proper hotkeys. I'm just the next person in line conducting science with it still while working on Unreal related stuff.

I don't like JASS all that much and didn't feel like adjusting to Lua being familiar with C++/C#. So I went looking for other options and landed on [WCSharp](https://github.com/Orden4/WCSharp) based on the [template](https://github.com/Orden4/WCSharp/wiki/WCSharp-template) which was nice being more familiar. There's been some gotcha's but it's worked pretty well so far.

The source is a bit of a mess and I've been struggling to organize it as a benefit to working on it and not for the sake of it.

I'm strictly against the idea of protecting maps. It's a bad practice. After the return bug was fixed, a lot of maps are just dead. Maybe something in here will be useful to you? If it is I wouldn't mind hearing about it or being credited.
